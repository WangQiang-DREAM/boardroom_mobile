## 会议室推荐预约系统 

### 项目说明
 * 该项目的主要用途是提供公司会议室的预约状态查看和在线预约功能 

### 项目环境
 * 项目基于 node.js 版本为6.11及以上

### 项目安装
 1. 进入该项目目录 执行 npm install 安装项目依赖
 2. 执行 npm start 可以开启开发环境，端口为3030

### 项目打包
 * 项目开发完成后 可以执行 npm run build 进行项目打包
 * 若没有问题 可以直接将 build 目录交给服务端部署

### 技术栈说明
 * 该项目使用 react 进行单页的开发
 * 使用 mobx 进行数据管理
 * 使用 antd-mobile 作为UI库
